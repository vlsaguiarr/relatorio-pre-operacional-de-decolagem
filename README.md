RELATÓRIO PRÉ OPERACIONAL DE DECOLAGEM

Este relatório tem como objetivo principal, a análise de dados e parâmetros, para definir se o foguete esta 'PRONTO PARA DECOLAR' ou se faz necessário 'ABORTAR DECOLAGEM'.

Gerando um dataset com números ficticios, o projeto irá extrair informações como temperatura externa, pressão dos tanques e etc... comparar as informações por meio de um algoritmo de validação e tomar uma decisão, caso algum parâmetro esteja fora do ideal, a saída para este grupo será 'ABORTAR DECOLAGEM', e caso contrário, 'PRONTO PARA DECOLAR', também traz a informação de qual condição não foi atendida para a decisão tomada e algumas informações extras incluindo informações sobre energia, conseguimos também calcular a autonomia inicial da decolagem.

Este projeto conta com uma análise realizada pela IA Gemini informando sobre possíveis Riscos e Anomalias que possam ser corrigidos, além de uma análise bruta dos dados.
